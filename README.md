## 🚀 نحوهٔ نصب

### پیش‌نیازها
- Python 3.9 یا بالاتر
- pip (مدیریت بسته‌های Python)
- توکن ربات تلگرام
- کلیدهای API Toobit

### مرحلهٔ 1: کلون‌کردن یا دانلود
```bash
# اگر از git استفاده می‌کنید:
git clone https://github.com/yourusername/toobit-bot.git
cd toobit-bot

# یا فایل‌ها را دانلود کنید
```

### مرحلهٔ 2: نصب وابستگی‌ها
```bash
pip install -r requirements.txt
```

### مرحلهٔ 3: تنظیم فایل .env
```bash
# فایل .env.example را کپی کنید
cp .env.example .env

# سپس با ویرایشگری مورد نظر باز کنید و مقادیر را پر کنید
nano .env
# یا
vim .env
```

### مرحلهٔ 4: دریافت کلیدهای لازم

#### 🤖 توکن تلگرام
1. به [@BotFather](https://t.me/BotFather) بروید
2. دستور `/newbot` را ارسال کنید
3. نام و نام‌کاربری ربات را انتخاب کنید
4. توکن را کپی کنید و در `.env` قرار دهید

#### 💱 کلیدهای Toobit API
1. به [Toobit](https://www.toobit.com) بروید
2. به حساب خود لاگین کنید
3. به بخش [API Management](https://www.toobit.com/account/api-management) بروید
4. کلید API جدید ایجاد کنید
5. `API Key` و `Secret Key` را در `.env` قرار دهید

#### 💳 درگاه‌های پرداخت
- **Zarinpal**: https://panel.zarinpal.com
- **PayPal**: https://developer.paypal.com
- **Stripe**: https://dashboard.stripe.com

### مرحلهٔ 5: اجرای ربات
```bash
python main_bot.py
```

اگر همه چیز درست باشد، باید ببینید:
```
✅ ربات با موفقیت شروع شد
🚀 ربات در حال اجراست...
```

## 📖 دستورات ربات

### دستورات اصلی
| دستور | توضیح |
|--------|---------|
| `/start` | منوی اصلی و ترحیب |
| `/help` | نمایش راهنما |
| `/wallet` | مدیریت کیف‌پول |
| `/trading` | مرکز ترید |
| `/prices` | قیمت‌های لحظه‌ای |
| `/orders` | مدیریت سفارشات |
| `/subscription` | مدیریت اشتراک‌ها |
| `/profile` | نمایش پروفایل |
| `/settings` | تنظیمات کاربر |

### مثال‌های استفاده

#### 💰 افزودن موجودی
```
/start → 💰 کیف‌پول → 💵 افزودن موجودی → مبلغ را وارد کنید
```

#### 🛒 خرید ارز
```
/start → 🛒 ترید → 🟢 خرید → نماد (مثل BTCUSDT) → مقدار → قیمت
```

#### ⭐ خریج اشتراک
```
/start → ⭐ اشتراک → تیر → مدت (ماهانه/سالانه) → روش پرداخت
```
