# 🎯 خلاصهٔ ربات معاملاتی Toobit

## 📌 بررسی نهایی

این ربات یک **سیستم کامل و حرفه‌ای** برای معامله در صرافی Toobit است که شامل:

### ✨ ویژگی‌های انجام‌شده

✅ **سیستم کیف‌پول کاملِ**
- کیف‌پول‌های چند‌ارزی
- اضافه‌کردن موجودی
- برداشت و انتقال
- تاریخچه تراکنش‌ها

✅ **سیستم اشتراک‌های 6‌سطحی**
- Free (رایگان)
- Lite (50,000 ت)
- Pro (150,000 ت)
- Professional (300,000 ت)
- Gold (500,000 ت)
- Diamond (1,000,000 ت)

✅ **درگاه‌های پرداخت**
- USDT Wallet مستقیم
- Zarinpal (بانکی ایرانی)
- PayPal (بین‌المللی)
- Stripe و سایر ها
- تاریخچه پرداخت‌ها

✅ **سیستم ترید پیشرفته**
- خرید و فروش مستقیم
- قیمت‌های لحظه‌ای
- مدیریت سفارشات
- تاریخچه معاملات
- آمار پورتفولیو

✅ **رابط کاربری بهتر**
- منوهای زیبا و سازمان‌یافته
- inline keyboards
- state management
- پیام‌های واضح و خوب‌فرمت

✅ **سیستم امنیتی**
- رمزنگاری
- محدود‌سازی سرعت
- اعتبارسنجی
- لاگینگ تمام فعالیت‌ها

✅ **پنل مدیریتی**
- آمار سیستم
- مدیریت کاربران
- مسدود/رفع مسدودی
- گزارش‌های مالی
- ابزار debugging

✅ **توابع کمکی گسترده**
- تبدیل ارز
- قالب‌بندی اعداد
- تاریخ و زمان
- صفحه‌بندی
- کش‌کردن

## 📊 جداول دیتابیس

```sql
users
  ├─ user_id (PRIMARY KEY)
  ├─ username
  ├─ first_name, last_name
  ├─ tier (free, lite, pro, professional, gold, diamond)
  ├─ wallet_balance
  ├─ created_at
  └─ is_banned

wallets
  ├─ wallet_id (PRIMARY KEY)
  ├─ user_id (FOREIGN KEY)
  ├─ currency (USDT, USD, IRR)
  ├─ balance
  └─ updated_at

transactions
  ├─ transaction_id
  ├─ user_id
  ├─ type (deposit, withdraw, trade, etc)
  ├─ amount
  ├─ currency
  ├─ status
  └─ created_at

subscriptions
  ├─ subscription_id
  ├─ user_id
  ├─ tier
  ├─ price
  ├─ start_date, end_date
  ├─ is_active
  └─ auto_renew

orders
  ├─ order_id
  ├─ user_id
  ├─ symbol (BTCUSDT, ETHUSDT, etc)
  ├─ side (BUY, SELL)
  ├─ quantity
  ├─ price
  └─ status

payments
  ├─ payment_id
  ├─ user_id
  ├─ amount
  ├─ method (usdt_wallet, bank_transfer, credit_card)
  ├─ status
  └─ created_at
```

## 🔄 جریان کار (Workflow)

### خرید اشتراک
```
کاربر → /start → ⭐ اشتراک → تیر → مدت → روش پرداخت
  ↓
Payment Gateway
  ↓
تأیید → Update Wallet → Create Subscription → بروزرسانی Tier
```

### خرید ارز
```
کاربر → /start → 🛒 ترید → 🟢 خرید → نماد → مقدار → قیمت
  ↓
Validation (بررسی تیر و محدودیت‌ها)
  ↓
Toobit API
  ↓
Order Created → تاریخچه → اطلاع‌رسانی
```

### افزودن موجودی
```
کاربر → 💰 کیف‌پول → 💵 افزودن → مبلغ → روش پرداخت
  ↓
Payment Gateway
  ↓
درخواست پرداخت (صبر برای تأیید)
  ↓
تأیید → Update Wallet → Transaction Log
```

## 🐛 حل مشکلات متداول

### مشکل: "Token invalid"
```
✓ توکن را دوبار چک کنید
✓ از @BotFather توکن جدید بگیرید
✓ فایل .env را مجدد لود کنید
```

### مشکل: "Database locked"
```
✓ تمام نمونه‌های ربات را بسته‌اند
✓ دیتابیس را بازگشایی کنید
✓ toobit_bot.db را حذف کنید (اخطار: داده‌ها حذف شوند)
```

### مشکل: "API Rate Limit"
```
✓ درخواست‌ها را کند کنید
✓ cache را استفاده کنید
✓ فاصلهٔ زمانی را اضافه کنید
```
